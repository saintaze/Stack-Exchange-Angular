# Stack-Exchange-Angular
An app in which you can search questions on stack exchange.

### Features
+ Responsive Design
+ Clean UI
+ Pagination
+ Master/Detail View

##### Built With
+ HTML5
+ SASS
+ Angular
+ Bootstrap
+ Stack Exchange API (Search)

### Getting Started
clone repo

`cd` into folder and run `npm install`

run `ng serve`

# Live Version
